table sorter - cross browser jQuery plugin

After include the relevant files in the <HEAD> of your document, initialise plugin on document.ready:

    $(document).ready(function()
    {
        $('#examples table').tableSorter();
    });

For example, we can stop row dragging for a third table by using

    $('#drag-table-3').tableSorter('stopRowsDrag');

If the table has no <TH> tag, the event to move columns is not possible.

The Plugin uses CSS:

    .empty-object {background:#034 !important; color:#034 !important;}

To finish editing cell press esc or enter when edit mode on.
